Joke Generator App

To read the document of using this app url will be: "localhost:3000/"

For the random Joke

To Get a random joke Api url will be :"localhost:3000/random"

For the random joke you have to type url localhost:3000/random.Here random is the ennpoint of the created Api.Localhost is computer which grabs the request from the server and get a joke from the data of json format from the backend.And the port number is 3000

If you want to filter the jokes by id you can use url like :"localhost:3000/jokes/(Here will be the id of the joke):id

And if you want to filter the jokes by the type ApiUrl willbe :"localhost:3000/jokes/filter?type=(Here will be the type of the joke)

To Add a new joke ApiUrl will be:"localhost:3000/jokes

By hitting a post request you can add a new joke in the data of the joke

To replace a new joke ApiUrl willbe localhost:3000/jokes/(integer):id

By hitting that routes you can replace a new joke completely route should PUT .Works with the id entered into the url
